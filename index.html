Images With Attribution From XMP

How Does This Work?

We use JavaScript to extract the XMP metadata from each image file, which requires downloading the file a second time after the browser downloaded it to display it intitally.

We then format up the XMP metadata into an HTML+RDFa attribution block, and add a button to copy that attribution information to the clipboard (where supported).

The copy to clipboard is the most modern feature - the rest of the features are from HTML5 which is supported by all recent browsers. Tested on recent builds of Firefox and Chrome.
